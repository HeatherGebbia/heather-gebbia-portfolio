Customer Agreement Web Application

The Customer Agreement Web Application is a secure, web-based tool for managing customer agreements. Users can view and sign agreements efficiently, while Admin can customize questionnaires and monitor completed agreements. This project was originally developed in VB.NET Web Forms and has been rewritten in C# Web Forms to modernize the code. I am now actively working on migrating the application to .NET Core with Razor Pages as part of my modernization journey.

C# Web Forms Version

This version is fully functional and deployed to Azure. It represents the first modernization phase from VB.NET Web Forms to C# Web Forms.

.NET Core Razor Pages Version (Work in Progress)

This is an in-progress modernization project where I am rebuilding the application using .NET Core Razor Pages. The goal is to take advantage of cross-platform support, dependency injection, built-in identity, and other modern .NET features.
