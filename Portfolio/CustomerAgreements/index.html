Customer Agreement Web Application

The Customer Agreement Web Application is a secure, web-based tool for managing customer agreements. Users can view and sign agreements efficiently, while Admin can customize questionnaires and monitor completed customer agreements. This project was originally developed in VB.NET Web Forms and has been rewritten in C# Web Forms to modernize the code and improve maintainability.

Live Demo View Source Code

Key Features

  • Secure login with Microsoft authentication (currently disabled for demonstration purposes)
  • Dashboard to view signed agreements
  • Fully customizable Questionnaires for Customers
  • Dynamic forms for capturing customer agreements
  • Integrated with SQL Server for data storage

Technologies Used

C#, ASP.NET Web Forms, SQL Server, Microsoft Azure App Service, HTML, CSS, JavaScript, jQuery
